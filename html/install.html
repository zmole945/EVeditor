Install Eclipse and Eclipse Verilog editor plug-in

Install Java runtime or Java development kit (JDK)

Eclipse 3.0 needs 1.4.1 or later version of Java. You can download from Java 2 SDK Standard Edition 1.4.2 web page

Install Eclipse 3.0

You can download Eclipse 3.0 from Eclipse download page. I recommend Eclipse SDK instead of Runtime Binary because I am testing on Eclipse SDK. The Eclipse SDK depends on platform. You have to select eclipse-SDK-<platform>.zip, such as eclipse-SDK-win32.zip on Windows.

After downloading, you can simply unzip to any directory where you like. Then you can start eclipse from your installed directory.

Install Eclipse Verilog editor

You should download net.sourceforge.veditor_0.2.0.zip. Then you should unzip it to Eclipse plugins directory. If you installed Eclipse in c:\eclipse, the plugins directory is c:\eclipse\plugins. You will see many plug-ins in it, such as "org.eclipse.ant.core_3.0.0". Eclipse Verilog editor is installed correctly if you see net.sourceforge.veditor_0.2.0 in plugins directory. -- It is same manner with other Eclipse plug-in.

NOTE: Please check Window->Preference after installing the plug-in. If you cannot see "Verilog Editor" directory in the Preference dialog, you meet a bug of Eclipse 3.0. You have to do this to avoid the bug.
  1. Exit Eclipse.
  2. Delete or rename "configuration" directory. It is in Eclipse installation directory, e.g. c:\eclipse\configuration.
  3. Start Eclipse and check Window->Preference.
If you need source code of Eclipse Verilog editor, You can download net.sourceforge.veditor_source_0.2.0.zip
