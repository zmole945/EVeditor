Start Verilog project

Start new project

Select "File"->"New"->"Project" or "New"->"Project" on Navigator
Check "Show All Wizards" in Eclipse 3.0M8.
Select Simple Project and push "Next".
Enter Project name and push "Finish"
Unfortunately I have not implement "Verilog Project" yet.

Import existing directory tree to Eclipse

You have to prepare outside of Eclipse.
Make this ".project" file. You can name Project Name which you want.
<?xml version="1.0" encoding="UTF-8"?>
<projectDescription>
<name>Project Name</name>
<comment></comment>
<projects>
</projects>
</projectDescription>
Copy .project to top of directory tree which you want to import.
Return to Eclipse.
Select "File"->"Import" or "Import" on Navigator.
Select "Existing Project into Workspace" and push "Next".
Enter directory name which has .project file as "Project contents" and push "Finish".
