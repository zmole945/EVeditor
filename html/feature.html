How to use features

  • Go to matching bracket

    Select "Navigate"->"Go to"->"Matching Bracket" menu.
    Then cursor position goes to opposite bracket of "[]", "{}", "()" and "begin/end"

  • Contents outline view

    Select "Window"->"Show View"->"Outline".
    The cursor position goes to the outline element with double clicking.

  • Open declaration

    Go to module declaration of selected word by searching a project hierarchy.
    It is valid when you defined a same module name with file name, e.g. module "adder" is defined in file "adder.v".
    Select "Navigate"->"Open Declaration" menu, "Open Declaration" in popup menu of editor page.

  • Content assist

    Enter a few character and enter Ctrl+space or "Edit"->"Content Assist" menu.
    Select word from a proposal list.

    The proposal list may include code templates, signals and module instances, which depend on Verilog context.

  • Module Hierarchy

    Select "Window"->"Show View"->"Other..." menu, then select "Module Hierarchy" in "Eclipse Verilog Editor" category.

    You can open editor and go to declaration by double clicking on Module Hierarchy view.

    You can open popup menu on Module Hierarchy view by right button clicking. The menu show "copy text" and "copy hierarchy". The "copy text" copies module list to clipboard simply. The "copy hierarchy" copies hierarchical list.
